# Torat Moshe Institute Website

This project is a website for the **Torat Moshe Institute**, which deals with ancient manuscripts. The system provides a user-friendly interface for proofreading, managing user permissions, and facilitating collaboration. Additionally, the website includes an AI-powered feature for transcribing manuscripts.

## Project Overview

This repository includes three main components:
- **Client (React)**: A responsive web interface for users to interact with the system.
- **Server (Node.js)**: Backend logic and API to handle requests, authentication, and communication with the AI component.
- **AI (Python)**: A module responsible for handling the AI-based transcription of manuscripts.
